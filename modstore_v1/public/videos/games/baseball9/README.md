# Baseball 9 Unlimited Gameplay Videos

Place your Baseball 9 gameplay videos in this folder using the following naming convention:

- `baseball9_1.mp4` - Enhanced Gameplay (showing unlimited gems/stars)
- `baseball9_2.mp4` - Premium Features (showing maxed-out players/premium uniforms)
- `baseball9_3.mp4` - Secret Tricks (showing special mod features)

## Video Requirements

- Format: MP4
- Resolution: 1080p (1920x1080) preferred
- Duration: 30-60 seconds for TikTok optimization
- Content: Showcase modded features clearly
- Add ModStore watermark/logo overlay

## TikTok Content Tips

- Start with exciting gameplay highlights
- Show before/after of player stats upgrades
- Focus on one key mod feature per video (unlimited gems, maxed stats, premium equipment)
- End with a clear call-to-action to download from ModStore

When creating your TikTok content, remember to use the video captions to explain the key benefits of the mod and how to download it from ModStore.

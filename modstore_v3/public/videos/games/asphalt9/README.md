# Asphalt 9 Legends Mod Gameplay Videos

Place your Asphalt 9 gameplay videos in this folder using the following naming convention:

- `asphalt1.mp4` - Enhanced Gameplay (showing unlimited money/tokens)
- `asphalt2.mp4` - Premium Features (showing unlocked cars/upgrades)
- `asphalt3.mp4` - Secret Tricks (showing special mod features)

## Video Requirements

- Format: MP4
- Resolution: 1080p (1920x1080) preferred
- Duration: 30-60 seconds for TikTok optimization
- Content: Showcase modded features clearly
- Add ModStore watermark/logo overlay

## TikTok Content Tips

- Start with exciting race highlights or car showcase
- Show before/after of car upgrades or unlocks
- Focus on one key mod feature per video (unlimited money, premium cars, max upgrades)
- End with a clear call-to-action to download from ModStore

When creating your TikTok content, remember to use the video captions to explain the key benefits of the mod and how to download it from ModStore.
